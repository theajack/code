Jcode.js
基于Jetterjs的代码编辑器的js库
View project on github
功能
这是一款基于Jetterjs的代码编辑器的js库, 只需在html中引入一个(或两个)script标签, 就可以在您的html文件中插入代码编辑器
使用方便简单
支持缩进缩退
功能强大
下载
引用： <script src="http://www.theajack.com/cnchar/assets/js/jcode.min.js"></script>
若是您没有使用jetter,您还需引用： <script src="http://www.theajack.com/jetterjs/assets/js/jetter.min.js"></script>
